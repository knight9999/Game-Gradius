# Game

## Description

純粋にJavascript+CANVASで実装しています。 

以下のゲームを楽しめます。
1. モアイステージ
2. 火山ステージ
3. スピードステージ
4. クリスタルステージ
5. デバッグ用ステージ
 
対応端末
- PC（Chrome、Safari）
- iOS（iPhone,iPad over 11）

操作説明 
- [A]→ショット＋ミサイル（連射付き）
- [B]→装備
- [P]→一時停止
- [S]→ステージ選択に戻る
- [R]→再度やり直し
